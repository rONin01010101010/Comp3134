Stored XSS Issue Analysis
=========================

Issue Identified:
-----------------
The file storexss.php reads the contents of storedxss.txt and outputs them directly to the browser
using echo, without any sanitization or encoding. This means that any HTML or JavaScript stored
in the text file is executed by the browser as live code — not displayed as plain text.

What Happens:
-------------
The content stored in storedxss.txt is:
    <script>document.location.href="/directory_traversal_part1.php"</script>

When storexss.php is visited, the browser receives and executes this script tag. The JavaScript
immediately redirects the user to /directory_traversal_part1.php without their knowledge or consent.

Why This Is Dangerous:
----------------------
1. Malicious Redirect: Any visitor to storexss.php is silently redirected to an attacker-controlled
   or unintended page (in this case, a directory traversal script).

2. Persistent Attack: Unlike reflected XSS (which requires the victim to click a crafted URL),
   stored XSS is persistent — the malicious payload is stored in a file or database and executes
   for EVERY user who visits the page, without any interaction required.

3. Privilege Escalation Risk: If an admin visits the page, the script executes with their session,
   potentially granting attackers elevated access.

4. Chained Attacks: The redirect target (directory_traversal_part1.php) could itself expose
   sensitive server files, making this a two-stage attack.

5. No Input Validation: The application trusts the stored file completely and makes no attempt
   to treat its contents as data rather than code.

How to Fix:
-----------
- Use htmlspecialchars() or htmlentities() when echoing any content from external sources:
      echo htmlspecialchars($contents, ENT_QUOTES, 'UTF-8');
- Treat all stored content as untrusted data, regardless of where it came from.
- Implement a Content Security Policy (CSP) header to restrict script execution.
- Never store raw user-supplied or externally-sourced HTML/JS without sanitization.
